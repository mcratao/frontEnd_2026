Fundamentos de HTML - Elementos Semânticos

O que são elementos semânticos?

São tags que possuem significado, ajudando a organizar melhor o conteúdo.

Principais Elementos Semânticos:

  • <header> - Cabeçalho
  • <nav> - Navegação
  • <main> - Conteúdo principal
  • <section> - Seção
  • <article> - Artigo
  • <footer> - Rodapé

Exemplo:

<header>
  <h1>Título</h1>
</header>

<main>
  <section>
    Conteúdo
  </section>
</main>

<footer>
  Rodapé
</footer>
