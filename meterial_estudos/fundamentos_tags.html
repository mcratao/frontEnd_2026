Fundamentos de HTML - Conhecendo as Tags

O que é uma tag?

Tags são comandos usados para estruturar uma página HTML.

Exemplos de Tags:

  • <html> - Estrutura principal
  • <head> - Informações da página
  • <body> - Conteúdo visível
  • <h1> - Título principal
  • <p> - Parágrafo
  • <a> - Link

Estrutura básica:

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  </body>
</html>
